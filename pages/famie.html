Famie and FamKid (Parental Control Applications)

Technologies: Flutter, Dart, Node.js, MongoDB Atlas, Heroku

Developed as part of the thesis "Optimizing Screen Time Management For Children's Devices: Leveraging Token Bucket And Time-Based Algorithms In A Famie Parental Control System," Famie (for parents) and FamKid (for children) are mobile apps that promote healthy digital habits using token bucket and time-based algorithms for screen time control, with secure device pairing via QR code scanning..
